Vue.js的小学生笔记

by 江矿宝宝.

2017-05-27

引入

本地还是网络导入都行

<script src='https://unpkg.com/vue@2.1.10/dist/vue.js'></script>

数据绑定

实现数据绑定的几个步骤

  • 创建vuejs实例
  • 划分vuejs势力范围
  • 向势力范围填充数据
<div id="demo">  
<!--vuejs中的变量格式-->  
        <h1>Hello {{msg}}</h1>
</div>  
<script>  
<!--创建vue对象-->  
new Vue({  
<!--el指定为#demo的dom对象-->  
  el:'#demo',
<!--msg变量赋值为haha-->  
  data:{
    msg:'haha'
    }
  })
</script>

双向绑定

vuejs通过v-model属性实现表单控件元素的双向绑定，只能用于input、select、textarea、components

<div id="demo">  
<!--vuejs中的变量格式-->  
        <h1>屠龙宝刀,{{msg}}</h1>
        <input type="text" v-model="msg">
</div>  
<script>  
<!--创建vue对象-->  
new Vue({  
<!--el指定为#demo的dom对象-->  
  el:'#demo',
<!--msg变量赋值-->  
  data:{
    msg:'点击就送'
    }
  })
</script>

你会看到在输入框输完文字后，{{msg}}里面的文字也会跟着变，即所谓的双向绑定。

内容绑定

vuejs通过v-text属性实现整个标签的内容绑定

实际上和引用vue变量的效果是一样的，来看实例

<div id="demo">  
<!--vuejs中的变量格式-->  
        <div>{{msg}}</div>
        <div v-text='haha'></div>
        <input type="text" v-model="haha">
</div>  
<script>  
<!--创建vue对象-->  
new Vue({  
<!--el指定为#demo的dom对象-->  
  el:'#demo',
<!--msg变量赋值为haha-->  
  data:{
    msg:'v-text',
    haha:'wtf',
    }
  })
</script>

可以看到原来的div内并没有文本，但v-text会实现整个标签的内容绑定
